# 価格適合性と分析粒度の評価

## 結論
- 単店舗12,000円は成立しやすい
- 多店舗10アカウント10万円は、分析軸が揃えば妥当

## 現状の分析粒度でできていること
- 日次/週次/月次の売上推移
- メニュー別売上ランキング
- 時間帯別の収益傾向
- 前年同月比（簡易）

## 弱い/未実装に見える点
- スタッフ別の収益貢献（実装が空に見える）
- 利益/粗利（コスト分析が固定値）
- 顧客リピート/初回率/離脱
- 予約チャネル別比較（ホットペッパー vs 自社）

## 価格の説得力を上げる優先軸
1. スタッフ別・担当別の貢献分析
2. チャネル別の集客/売上比較
3. 粗利/利益ベースの収益分析
